# 🚗 AI-Based Embedded Driver Monitoring System

## 📌 Overview
This project implements an AI-based driver monitoring system integrated with embedded sensors and Verilog-based hardware alert logic.

## 🎯 Features
- Face Detection using OpenCV
- Drowsiness Detection
- Alcohol Detection using MQ-3 Sensor
- Hardware Alert Logic (Verilog)
- Functional Verification using Testbench

## 🏗️ Architecture
Camera + Sensors → Raspberry Pi → AI Processing → Verilog Alert Logic → Buzzer Alert

## 💻 Technologies
- Python (OpenCV)
- Raspberry Pi
- Verilog HDL
- GPIO Sensors

## 🧪 Verification
- Testbench simulation
- Waveform generation (VCD)
